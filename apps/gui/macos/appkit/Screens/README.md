# Screens

FACMAN-APPKIT-SHELL-01 starts with the same narrow screen set as the WinForms
proof:

- Dashboard
- Doctor
- Installs
- Instances
- Launch Plan
- Diagnostics
- Settings/About

These screens present command IDs and backend results. They do not own Factorio
discovery, modset resolution, save backup/export/import behavior, server
execution, developer execution, or setup mutation.

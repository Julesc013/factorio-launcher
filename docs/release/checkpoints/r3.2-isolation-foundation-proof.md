## Stretch B: Bounded Diagnostics Traversal Foundation

`FACMAN-DIAGNOSTIC-BOUNDED-TRAVERSAL-03` replaces the dormant instance-wide
recursive walk with deterministic allowlisted traversal. The policy refuses
links and reparse points through the shared path-safety primitive and enforces
maximum depth, file count, per-file size, total selected bytes, and elapsed
time. Every omitted path receives a structured reason in
`factorio.diagnostic_traversal_report.v1`.

The redactor now treats `.ini` as structured input and fails closed on malformed
sections or key/value lines, matching its existing malformed-JSON behavior.
Native proof covers spaces, Unicode, long paths, overlapping allowlists, depth,
count, size, time, and escape behavior. Python regressions include real Windows
junction refusal and prove malformed sensitive INI does not escape. Both export
entry points remain quarantined.

This is a foundation promotion only. `facman diagnostics export` and doctor
bundle output still return `diagnostic_export_not_safe`; no general diagnostic
sanitization or export claim is promoted.

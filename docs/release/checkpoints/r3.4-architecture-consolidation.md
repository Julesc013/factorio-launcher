## Target promotion hold

The local host cannot produce the required Ubuntu 24.04 ASan/UBSan,
libFuzzer, clang-tidy, coverage, and zero-skip Linux package result. Its WSL
environment has no C/C++ toolchain, and the installed Windows GCC distribution
has no `libasan` or `libubsan`. It also cannot produce the macOS 15 Intel native
CLI/package or AppKit compile result.

Those lanes remain planned as
`FACMAN-R3.4-TARGET-RUNNER-PROMOTION-01`. They require an authorized push or
equivalent runner handoff. Previous R3.3 target evidence is not reused as fresh
R3.4 proof.

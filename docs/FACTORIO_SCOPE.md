# Factorio Scope

In scope:

- read-only install discovery
- manual and portable install registration
- isolated instances
- per-instance config, mods, saves, scenarios, script output, logs, crash files
- launch plans using Factorio command-line options
- local ZIP mod import and modset lockfiles
- save backup/export workflows
- headless server workflows
- modder diagnostics

Out of scope for early versions:

- bundled Factorio binaries
- DRM bypass
- official branding
- GUI-only behavior
- repair or uninstall of foreign-owned installs

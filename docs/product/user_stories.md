# User Stories

- As a player, I can keep several complete Factorio instances isolated so one
  version, modpack, profile, account context, or setting does not pollute
  another, while each instance may contain multiple worlds.
- As a player, I can select an instance and open Factorio's normal main menu
  without being forced to choose or directly load a save.
- As a player, I can model any desired version, content, mod, save, profile,
  account-binding, and environment combination and receive an honest readiness
  result even when the combination cannot be executed.
- As a player, I can clone a known-good instance before changing its version,
  modset, preset, or material profile.
- As a player, I can use Continue, Load Save, New Game, Map Editor, Join Server,
  and server/development modes only as explicit launch alternatives.
- As a Steam user, I can discover my Steam install without the launcher taking
  ownership of it.
- As a modded player, I can lock a modset and reproduce it later.
- As a server host, I can export a server pack with redacted secrets.
- As a modder, I can generate diagnostics and bug-report bundles.

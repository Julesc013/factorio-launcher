## Permanent Wave Boundaries

- no real Factorio execution or automated operator verdict;
- no `run.execute`, Safe beta, setup mutation, networking, credentials,
  dynamic plugins, server execution, signing, release, or publication;
- one archive dependency maximum, admitted only after license, provenance,
  checksum, notice, lock, SBOM, and compile evidence;
- one structured commit per independently green WorkUnit;
- diagnostic export remains unavailable unless its complete proof passes.

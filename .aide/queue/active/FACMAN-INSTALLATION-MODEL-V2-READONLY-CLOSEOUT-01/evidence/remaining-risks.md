# Remaining risks

- Reconciliation output is advisory and plan-only. No operation permit or
  provider apply route exists.
- Authenticated source inspection, exact staged source closure, and source
  acquisition remain unimplemented.
- Create, adopt, repair, move, update or downgrade, reinstall, detach,
  uninstall, and mutation recovery are transferred to
  `FACMAN-MANAGED-INSTALL-RECONCILIATION-01`.
- The compatible v1 installation reference remains the persisted record;
  model v2 is not a workspace migration or new authority source.
- Filesystem, provider, setup-state, executable, and verification identities
  that are not observed remain explicit per-operation revalidation
  requirements.
- Real Factorio execution, `OperationPermit`, instance preparation, network,
  credentials, host mutation, signing, publication, and Safe beta remain
  unavailable.
- The original `FACMAN-MULTI-VERSION-INSTALL-LIFECYCLE-01` objective included
  the deferred mutation lifecycle and is therefore superseded with
  `objective_complete=false`; this bounded closeout does not claim otherwise.
